# Analyse Accessibilité Réseau RTC

Projet géomatique — visualisation de l'accessibilité piétonne du réseau de bus RTC de Québec.

## Démarrage rapide

```powershell
# 1. Démarrer ORS (si besoin de recalculer des isochrones)
docker start ors-app

# 2. Lancer le site
cd site
python -m http.server 8081

# 3. Ouvrir http://localhost:8081
```
